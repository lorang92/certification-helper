You develop an HTML5 webpage that contains the following HTML markup: 

    <label for="loanTypes">Loan Types</label>
    <select name="loanTypes" id="loanTypes" multiple="multiple">
        <option>Secured</option>
        <option selected="selected">Unsecured</option>
        <option>Demand</option>
        <option selected="selected">Subsidized</option>
    </select>

You have the following requirements:

  • Retrieve the content for any OPTION element when the value of the selected attribute equals selected.
  • Add the content to a comma separated string.

You need to retrieve the required data. Which two code segments should you add to the webpage?
(Each correct answer presents a complete solution. Choose two.) 

  1.                 $("select option:checked").each(function () {
                    selectedValues += $(loanTypes).text() + ',';
                });

                    selectedValues += $(loanTypes).text() + ',';
                });
            
  3.                 $("select option:selected").each(function () {

                });
