WebSocket demo
--------------

To install, type the following in a command prompt:
   npm install

To start this websocket en http (client) servers, type the following in a command prompt:
   npm start

To test the websocket server is running, open a browser and enter this URL:  
   http://localhost:8888   (this should display index.html in the browser)

To run the http (client) server, open a browser and enter this URL:  
   http://localhost:8080

To close the servers again, type:
    ctrl+c (and choose Y, or type ctrl+c twice).
